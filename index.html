Python Sci-kit Learn RDKit Seaborn Pandas Numpy
The goal of this project was to learn and demonstrate concepts in machine learning with a novel project, instead of a cookie-cutter one found on kaggle. I generated 4 different molecular fingerprints for ~7000 compounds, tackled class imbalance, and trained models using each fingerprint as a single feature. I evaluated model performance for logistic regression, knn, and gradient boosting - based on validation data accuracy, AUC, and f1-scores.
